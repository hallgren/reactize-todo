reactize-todo
=============

todomvc in reactize


#### Running locally

1. Clone this repo

        $ git clone https://github.com/hallgren/reactize-todo.git

2. Install bundle dependencies

        $ bundle install

3. Run the server (change the port with the `-p` flag)

        $ bundle exec rackup

4. Visit the app: [http://localhost:9292](http://localhost:9292)
